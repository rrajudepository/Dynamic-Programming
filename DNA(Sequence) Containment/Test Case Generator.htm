Examples

  1. To generate a string consisting of characters "abcd" such that c occurs maximum number of times in the resulting string,

    In the Characters Allowed box, enter a string containing a,b,c,d in which c occurs more number of times than the rest.
    For example, to generate strings similar to
    cababccbcccbcddccbdccccbcdcccbcbacccccbcccccdcccab
    you may enter a string like abcccccd or bccccbcccda or cccccccccccbbddaa etc. in the Characters Allowed box.


  2. To generate a 2D maze containing . and # where . represents a free cell ,# represents a wall and . occurs more number of times than #.

    In the Characters Allowed box, enter a string which contains a large number of . compared to #.
    For example, to generate strings similar to
    .......#.
    ....#....
    ...#..#..
    #.....#..
    you may enter a string like #...##......###..... or ...#...#.... etc. in the Characters Allowed box.
